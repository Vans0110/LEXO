Что есть
Универсального iPhone-эмулятора для Windows нет
Официальный iOS Simulator существует только в Xcode на macOS
На Windows лучший предосмотр mobile UI — Android Emulator
Для Flutter это нормальный промежуточный шаг: Flutter официально поддерживает запуск на Android Emulator, а Android Studio официально даёт виртуальные устройства с разными размерами и конфигурациями экранов.

Не искать “эмулятор айфона на Windows”, а сделать так:

1. Поднять Android Emulator в размере, близком к iPhone

Это даст тебе:

вертикальный телефонный экран
проверку layout
проверку тапов
проверку навигации
проверку клавиатуры
проверку того, что интерфейс вообще не разваливается

Android Emulator — официальный инструмент Android Studio, и он как раз предназначен для тестов на виртуальных устройствах разных форм-факторов. Flutter тоже прямо поддерживает запуск на Android Emulator.

2. Делать первый Mobile UI именно там

То есть:

не пытаться увидеть “настоящий iPhone”

Итог

Тебе сейчас нужен не “эмулятор айфона”, а предпросмотр мобильного интерфейса.
Для Windows лучший путь — Android Studio Emulator: он бесплатный, официальный и подходит для Flutter mobile UI проверки до первой iOS сборки. Но iPhone он не заменяет: официальный iOS Simulator доступен только через Xcode на macOS.

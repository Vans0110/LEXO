Инициализация MVP-каркаса проекта

- В репозитории были только `MVP.txt` и `AGENTS.md`

## Что сделано

### 1. Создана базовая структура проекта

Добавлены каталоги и стартовые файлы:

- `engine/`
- `app/`
- `data/current_book/`
- `data/logs/`
- `.gitignore`

### 2. Реализован Python engine

Добавлены файлы:

- `engine/main.py`
- `engine/storage.py`
- `engine/text_utils.py`

Что реализовано:

- импорт `.txt` книги через `/books/import`
- разбиение текста на абзацы

- получение абзацев для reader
- lookup слова с нормализацией
- локальный cache для данных слова
- сохранение слов в список saved words
- stub-генерация аудио для слова и абзаца

Что заложено:

- Flutter SDK в текущем окружении отсутствует, поэтому desktop UI не проверен реальным запуском
- иконка runner пока не подключена; при необходимости можно добавить настоящую `.ico` позже
- при первом `flutter run` Flutter может сгенерировать дополнительные служебные файлы

## Дополнение за 2026-03-28

### Сессия

Подготовка запуска MVP одним кликом

### Что сделано

- добавлен bat-файл `run_lexo_mvp.bat`
- добавлен файл `RUN_MVP.md` с краткой инструкцией
- `README.md` дополнен упоминанием запуска через `.bat`

### Что делает `run_lexo_mvp.bat`

- проверяет `python` в `PATH`
- проверяет `flutter` в `PATH`
- при первом запуске делает `flutter pub get`
- запускает `python -m engine.main` в отдельном окне
- затем запускает `flutter run -d windows` для папки `app`
